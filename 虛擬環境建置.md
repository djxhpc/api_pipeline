## Step 1：解鎖腳本執行權限
如果終端機因安全性設定不讓開環境先輸入：
```bash
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process
```
## Step 2：安裝虛擬環境
```bash
pip install virtualenv
```
## Step 3：建立虛擬環境
```bash
python -m venv yourname
```
## Step ：啟用虛擬環境
```bash
.\yourname\Scripts\activate
```
## 退出環境
```bash
deactivate
```
